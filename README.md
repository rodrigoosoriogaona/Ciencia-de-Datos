#  House Prices – Predicting Housing Sale Prices

 Proyecto académico para aplicar técnicas de regresión, análisis de datos y machine learning utilizando el dataset *House Prices: Advanced Regression Techniques* de Kaggle.

##  Objetivo del Proyecto
Analizar los factores que influyen en el precio de venta de las viviendas y construir modelos predictivos para estimar su valor con mayor precisión.

---

##  Tareas del Proyecto

1️ **Análisis Exploratorio de Datos (EDA)**
- Limpieza y transformación de datos
- Visualización de variables relevantes
- Detección de outliers y valores faltantes

2️ **Construcción de Modelos Predictivos**
- Regresión Lineal
- Árboles de Decisión
- Random Forest (u otros modelos)

3️ **Evaluación y comparación de modelos**
- Métricas: RMSE, MAE, R²

---

 Kaggle — *House Prices: Advanced Regression Techniques*  
Dataset utilizado: `train.csv`
